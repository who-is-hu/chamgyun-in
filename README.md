## 프로젝트 중점 사항
- 유연한 설계
- 테스트 코드 작성
- 정확하게 학습한 기술을 사용하기
  
[wiki](https://github.com/who-is-hu/chamgyun-in/wiki)에서 고민의 흔적이나 과정을 볼 수 있습니다.
