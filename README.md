# chamgyun-in

일상의 사소한 고민에서 선택하지못하는 우리를 위해 준비한 프로젝트입니다.  
내 고민에 대한 해결방안을 선택지로 공유하고 **익명의 투표**를 받아 선택에 도움을 받아보세요.

## 프로젝트 중점 사항
- 유연한 설계
- 테스트 코드 작성
- 정확하게 학습한 기술을 사용하기
